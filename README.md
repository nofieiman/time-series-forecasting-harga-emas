# Time Series Forecasting Harga Emas Pegadaian Berdasarkan Nilai Tukar Rupiah

📈 Prediksi harga emas Pegadaian dengan pendekatan *Time Series Forecasting* berdasarkan variabel eksternal nilai tukar Rupiah terhadap USD.

## 🧠 Deskripsi Proyek

Proyek ini bertujuan untuk membangun model peramalan (*forecasting*) harga emas Pegadaian di Indonesia dengan mempertimbangkan pengaruh nilai tukar Rupiah. Model ini dapat digunakan oleh pelaku pasar, investor ritel, maupun lembaga keuangan untuk membantu pengambilan keputusan yang lebih baik dalam membeli atau menjual emas.

## 🛠️ Metodologi

Notebook ini mencakup beberapa tahapan penting:

1. **Pengumpulan dan Praproses Data**  
   - Sumber data harga emas dan nilai tukar
   - Pembersihan dan transformasi data

2. **Eksplorasi Data dan Visualisasi**  
   - Visualisasi tren harga emas dan nilai tukar
   - Korelasi antara variabel

3. **Model Time Series Forecasting**  
   - Model ARIMA / SARIMA / VAR
   - Evaluasi performa model
   - Peramalan harga emas berdasarkan data historis dan nilai tukar

4. **Visualisasi Hasil Peramalan**  
   - Grafik harga aktual vs prediksi
   - Plot residual dan error

## 📊 Tools & Library

- Python 3.x
- Pandas
- Numpy
- Matplotlib, Seaborn
- Statsmodels
- Scikit-learn
- Prophet / XGBoost jika dikembangkan lebih lanjut

## 🚀 Hasil dan Manfaat

Model ini memberikan insight terhadap potensi fluktuasi harga emas Pegadaian yang dipengaruhi oleh nilai tukar Rupiah. Proyek ini dapat dikembangkan lebih lanjut untuk:
- Integrasi dengan faktor eksternal lain (misalnya suku bunga BI, harga emas dunia)
- Pengembangan dashboard interaktif (misal: Streamlit)
- Otomatisasi prediksi harian

## 📌 Catatan

- Proyek ini hanya bertujuan untuk edukasi dan eksplorasi. Tidak dimaksudkan sebagai rekomendasi investasi.
- Akurasi model dapat bervariasi tergantung kualitas dan cakupan data.
